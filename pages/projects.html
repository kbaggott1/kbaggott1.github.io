Vending Machine

This project is a WPF application, it mimics a vending machine by have a keypad, screen for the keypad, a display with all the items you can purchase, and the ability to pay with "debit" or "cash". These items are stored using a csv file, which contains the name of the product as well and the amount in stock. There is also an option to save the current stock to the csv if for instance you purchased items and want to save the current amount of items in stock for the next time you run the application.

Aliens Attack

Aliens Attack is a video game made in C#. It was my first time ever learning/using object oriented programming in 2021. It is a game inspired off of space invaders where you control a space ship and have to fight off alien space ships to progress furthur into different levels with a boss fight at the end. Game progress is saved using an xml file and the highscore for each level is saved in the same file as well.
